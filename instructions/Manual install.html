Полуавтоматическая установка

Внимание! Полуавтоматическая установку стоит использовать только в том случае, если файлы .htaccess, style/Default/index.html и instruments/menu_items.php были изменены

В противном случае, воспользуйтесь автоматической установкой

Открываем файл .htaccess и после строчки

RewriteRule ^go/([^/]+)/?$ index.php?mode=$1 [L,NE]

Добавляем

RewriteRule ^go/gallery/p.([^/]+)/?$ index.php?mode=gallery&pid=$1 [L,NE]
RewriteRule ^go/gallery/img.([0-9]+)(\.[a-z]+)?/?$ index.php?mode=gallery&do=image&img=$1 [L,NE]
RewriteRule ^go/gallery/thumb.([0-9]+)(\.[a-z]+)?/?$ index.php?mode=gallery&do=thumb&img=$1 [L,NE]
RewriteRule ^go/gallery/([^/]+)/?$ index.php?mode=gallery&do=$1 [L,NE]

Открываем instruments/menu_items.php и перед строчкой

'options' => array (

Добавляем

'qx_gallery' => array (

'name' => 'Галерея',
'url' => Rewrite::GetURL('gallery'),
'parent_id' => -1,
'lvl' => -1,
'permission' => -1,
'active' => false,
'inner_html' => '',
),

Открываем style/Default/index.html и переносим строчку

<script src="<?php View::URL('js/jquery.js') ?>"></script> В верхнюю часть кода, прямо перед строчкой <?php echo $content_js ?>
  • Заливаем все файлы и папки(за исключением тех, которые изменили выше) в соответствующие каталоги
  • Переходим по адресу http://mysite.com/go/gallery/install
  • Следуем инструкциям установщика
